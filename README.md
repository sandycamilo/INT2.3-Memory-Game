
# *Memory Game*

 A color and sound memory game that helps you learn through play created with React.
 
 
# Built with 

 - Javascript 
 
 # Getting Started 
 
 To get a local copy up and running follow these simple steps: 
 
 1. Download or clone this repo 

```
$ git clone git@github.com:sandycamilo/INT2.3-Memory-Game.git

```
 2. cd into the project folder and open it in your chosen text editor
 
 ```
 $ cd INT2.3-Memory-Game
 
 ```
  2. Run app
 
 ```
 $ yarn 
 
 $ yarn start 
 
 ```
